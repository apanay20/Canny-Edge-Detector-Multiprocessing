## About The Project

This is an implementation of Canny Edge Detector algorithm using multiprocessing.

Author:
* Andreas Panayiotou

## Prerequisites

Below are shown the necessary python libraries to run the edge detector. 
* OpenCV
  ```sh
  pip install opencv-python
  ```
* Matplotlib
  ```sh
  pip install matplotlib
  ```
* NumPy
  ```sh
  pip install numpy
  ```
